# Publish Checklist (Konsortium)

## 1) Final Link Check
- Verify social links in `index.html` (founder Twitter/Instagram are still placeholders if not provided).
- Verify initiatives:
  - Konsortia Films -> `https://www.youtube.com/@konsortiafilms`
  - IRISE Capital -> `https://raajveer.com/`

## 2) Quick Local Smoke Test
- Open `index.html` in browser and confirm:
  - Founder images are cropped correctly.
  - All initiative links open correctly in new tab.
  - Contact and FAQ sections behave normally.

## 3) Deploy (Fastest: Netlify Drop)
- Go to `https://app.netlify.com/drop`
- Drag the project folder (`d:\Konsortium`) into the drop zone.

## 4) Domain + SEO
- Attach custom domain (if using `konsortium.org`).
- Confirm these files are served:
  - `/robots.txt`
  - `/sitemap.xml`
  - `/404.html`

## 5) Post-Launch
- Submit `https://konsortium.org/sitemap.xml` in Google Search Console.
- Share live URL in social profiles.
